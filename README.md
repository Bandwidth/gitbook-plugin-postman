# gitbook-plugin-postman
[Postman](https://www.getpostman.com/) for Gitbook. This loads the `_pm` script from postman to use for postman buttons

### How to use?

Add plugin to your `book.json`, then run `gitbook install`:

```json
{
    "plugins": ["postman"]
}
```
